# Argo

Argo 是一个前端组件管理系统。目前Argo可以提供如下能力：
* 前端组件模型定义功能
* 前端组件实例维护功能
* 前端页面和组件的关联功能
* 后端接口聚合功能

具体使用方式参考 **[WIKI](https://github.com/zhyea/argo/wiki)**
